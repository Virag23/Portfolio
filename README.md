## 🚀 Local Development Setup

To spin up this portfolio project locally, follow these instructions:

### Prerequisites
* [Node.js](https://nodejs.org/) installed (v18+ recommended)
* [npm](https://www.npmjs.com/) or another package manager

### Installation Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/<your-username>/Virag-Portfolio.git
   cd Virag-Portfolio
   ```

2. **Install all dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory:
   ```env
   # Frontend URL for CORS configuration
   CLIENT_URL=http://localhost:5173

   # Port configuration for the backend express server
   PORT=5000

   # SMTP Configuration (Nodemailer)
   SMTP_HOST=your-smtp-host
   SMTP_PORT=587
   SMTP_USER=your-smtp-username
   SMTP_PASS=your-smtp-password
   RECEIVER_EMAIL=viragsjain1975@gmail.com
   ```

4. **Run the local Development Servers:**
   * Run the React frontend:
     ```bash
     npm run dev
     ```
   * Run the Express backend server:
     ```bash
     npm run server
     ```
